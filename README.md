# React Big Calendar Project

## About the Project

This project is built using React Big Calendar library. It provides a modern and responsive calendar system for managing events and schedules.

## Features

* Monthly, weekly, and daily calendar views
* Event scheduling and management
* Responsive UI design
* Easy navigation between dates

## Technologies Used

* React.js
* JavaScript
* HTML
* CSS
* React Big Calendar

## How to Run

1. Install dependencies:
   npm install

2. Start the project:
   npm run storybook

## Purpose

This project helps in understanding how calendar-based scheduling systems work in real-world applications.
